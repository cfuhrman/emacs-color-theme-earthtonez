** Usage

   1) Copy the file =earth-tonez-theme.el= to your =~/.emacs.d=
      directory
   2) Add the following lines to =~/.emacs.d/init.el=

      #+BEGIN_SRC emacs-lisp
      (require 'color-theme-earth-tonez)
      #+END_SRC
   3) Either quit and restart Emacs or evaluate your
      =~/.emacs.d/init.el=
